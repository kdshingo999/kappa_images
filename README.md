# かっぱキャラクター画像生成ツール

OpenAI GPT Image 1.5を使用して、かっぱのキャラクター画像を生成するPythonスクリプトです。

## GPT Image 1.5について

GPT Image 1.5は2025年12月にリリースされたOpenAIの最新画像生成モデルです：

- **高速生成**: 従来のDALL-E 3より最大4倍速い生成速度
- **高精度**: より正確な指示追従と画像編集が可能
- **自然な結果**: より自然で高品質な画像を生成
- **テキスト描画**: 画像内のテキストレンダリングが改善
- **コスト効率**: DALL-E 3と比較して20%安いAPIクレジット

## 必要な環境

- Python 3.8以上
- OpenAI APIキー

## セットアップ

1. 依存パッケージのインストール:
```bash
pip install -r requirements.txt
```

2. OpenAI APIキーの設定（既に設定済みの場合はスキップ）:
```bash
export OPENAI_API_KEY="your-api-key-here"
```

## 使い方

基本的な実行:
```bash
python generate_kappa.py
```

スクリプトを実行すると:
- GPT Image 1.5がかっぱのキャラクター画像を生成
- 生成された画像を`generated_images/`ディレクトリにPNGファイルとして保存
- 画像情報（プロンプト、サイズ、画質など）をテキストファイルとして保存

## カスタマイズ

`generate_kappa.py`のコードを編集して、以下をカスタマイズできます:

- **プロンプト**: かっぱの見た目や雰囲気を変更
- **サイズ**: `1024x1024`, `1024x1792`, `1792x1024`
- **画質**: `standard` or `hd`

例:
```python
generate_kappa_image(
    prompt="かわいいかっぱのキャラクター。頭にお皿、甲羅を背負い、緑色の体。アニメ風のイラスト",
    size="1024x1024",
    quality="hd"
)
```

## 注意事項

- GPT Image 1.5は1回の実行につきAPIクレジットを消費します
- 生成された画像は`generated_images/`ディレクトリに直接PNGファイルとして保存されます
- 画像のサイズや画質によってAPIクレジットの消費量が変わります
